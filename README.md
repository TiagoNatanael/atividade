
# Gerenciador de Contas

Um sistema de gerenciamento de contas desenvolvido em Python. O projeto focado para o curso de back-end, este proheto segurança utilizando hash **SHA-256** para a proteção e armazenamento das informações para estudo proprio de cybersegurança.

## 🚀 Como configurar e rodar o projeto

### Pré-requisitos
* Python 3 instalado no sistema.

### Passo a Passo

1. Clone este repositório para a sua máquina:
git clone https://github.com/TiagoNatanael/atividade.git

2. Acesse a pasta do projeto:
cd atividade

3. (Recomendado) Crie e ative um ambiente virtual para isolar o projeto:
python -m venv venv
source venv/bin/activate

4. Execute o programa principal:
python "#tarefa_da_inova.py"
